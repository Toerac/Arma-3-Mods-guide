## 1.1 Настройка
###### Arma 3 Tools
Чтобы у вас открывались файлы нужными интрументами, нажмите `Misc > File association`

> [!attention] Иногда есть моды в которых нет селекшенов
> Проще всего заменить текстуру в моде
